# Diet Suggestion System

A machine learning-based diet recommendation system built using Streamlit and FastAPI.

## Features
- Personalized diet recommendations
- Custom food recommendations
- Machine learning-powered suggestions
- Interactive Streamlit UI

## Technologies
- Python
- FastAPI
- Streamlit
- Pandas
- Scikit-learn
- Docker

## Setup

pip install -r requirements.txt

streamlit run Streamlit_Frontend/Hello.py
